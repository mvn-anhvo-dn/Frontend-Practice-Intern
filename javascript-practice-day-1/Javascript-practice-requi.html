+ Null là một giá trị rỗng hoặc là không tồn tại, có thể được gán cho 1 biến để đại diện như không có giá trị

- What are the differences between `==` and `===`? Write an example for each case (if any)?

`==` : Chỉ so sánh về giá trị Ex : "123" == 123 //true 123 == 123 //true "123" == "123" //true

`===` : So sánh cả giá trị và kiểu dữ liệu Ex : "123" === 123 //false 123 === 123 //true "123" === "123" //true

- Give an example for each of the following methods in Javascript + map + filter + reduce + find + some

map : const myArr = [1,2,3,4]; const myArrTwo = myArr.map((value, index, array) => { return value * 2; }); console.log{myArrTwo}; //[2,4,6,8];

filter : const myArr = [1,2,3,4]; const myArrTwo = myArr.filter((value, index, array) => { return value % 2 === 0; }); console.log{myArrTwo}; //[2,4];

reduce : const myArr = [1,2,3,4]; const myArrTwo = myArr.reduce((acc, curValue, curIndex, array) => { return acc + curValue; }); const avg = myArrTwo / myArr.length; console.log{myArrTwo}; //2.5

find: const ages = [3, 10, 28, 20]; function checkAge(age) { return age > 20; } let age = ages.find(checkAge); console.log(age); // 28

some: const array = [1, 2, 3, 4, 5]; const even = (element) => element % 2 === 0; console.log(array.some(even)); // true
